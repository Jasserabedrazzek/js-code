DOM Utils - A Simple JavaScript Library

Easily create elements, apply styles, and add them to the DOM.

Download Library

Introduction

DOM Utils is a lightweight JavaScript library that simplifies creating and managing DOM elements dynamically.

Installation

To use this library, include the following script in your project:

import { createElement, applyStyles, appendToDOM } from 'https://create-page-with-js.netlify.app/dom-utils.js';

Usage

Examples: Creating a new element, styling it, and adding it to the page.

1. Simple Element Creation

This example creates a div with text content.

const newDiv = createElement({ tag: 'div', children: 'Hello, World!' }); appendToDOM({ element: newDiv, parent: 'body' });

2. Adding Attributes

This example adds attributes like id and class to the element.

const newDiv = createElement({ tag: 'div', attributes: { id: 'myDiv', class: 'container' }, children: 'This is a styled div!' }); applyStyles({ selector: '#myDiv', css: { backgroundColor: 'lightblue', padding: '20px', color: 'darkblue' } }); appendToDOM({ element: newDiv, parent: 'body' });

3. Creating a Button with a Click Event

This example creates a button and attaches a click event to it.

const button = createElement({ tag: 'button', children: 'Click Me!' }); button.onclick = function() { alert('Button clicked!'); }; appendToDOM({ element: button, parent: 'body' });

4. Nested Elements

This example creates a div containing a p element.

const newDiv = createElement({ tag: 'div' }); const newP = createElement({ tag: 'p', children: 'This is a nested paragraph inside a div!' }); newDiv.appendChild(newP); appendToDOM({ element: newDiv, parent: 'body' });

5. Dynamic List Creation

This example dynamically creates a list of items.

const items = ['Item 1', 'Item 2', 'Item 3']; const list = createElement({ tag: 'ul' }); items.forEach(item => { const listItem = createElement({ tag: 'li', children: item }); list.appendChild(listItem); }); appendToDOM({ element: list, parent: 'body' });
