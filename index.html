DOM Utils - A Simple JavaScript Library

Easily create elements, apply styles, and add them to the DOM.

Download Library

Introduction

DOM Utils is a lightweight JavaScript library that simplifies creating and managing DOM elements dynamically.

Installation

To use this library, include the following script in your project:

import { createElement, applyStyles, appendToDOM } from 'https://create-page-with-js.netlify.app/dom-utils.js';

Usage

Examples: Creating a new element, styling it, and adding it to the page.

1. Simple Element Creation

This example creates a div with text content.

const newDiv = createElement({ tag: 'div', children: 'Hello, World!' // The text content inside the div }); appendToDOM({ element: newDiv, // The created div element parent: 'body' // Append it to the body of the page });

Output: A simple `div` will appear with the text "Hello, World!" on the webpage.

2. Adding Attributes

This example adds attributes like id and class to the element.

const newDiv = createElement({ tag: 'div', attributes: { id: 'myDiv', class: 'container' }, // Adding id and class children: 'This is a styled div!' // The content inside the div }); applyStyles({ selector: '#myDiv', // Target the div with id 'myDiv' css: { backgroundColor: 'lightblue', padding: '20px', color: 'darkblue' } // Applying styles }); appendToDOM({ element: newDiv, parent: 'body' });

Output: A `div` with an ID of `myDiv` will be displayed with a light blue background, dark blue text, and padding. The text inside it will read "This is a styled div!"

3. Nesting Elements

This example creates a div containing a span element.

const newDiv = createElement({ tag: 'div', attributes: { id: 'parent' }, children: createElement({ tag: 'span', // Nested span inside div children: 'This is a nested span element' // Text content inside span }) }); applyStyles({ selector: '#parent', // Apply styles to the parent div css: { backgroundColor: 'lightgreen', padding: '20px' } }); appendToDOM({ element: newDiv, parent: 'body' });

Output: A `div` with the text "This is a nested span element" inside a nested `span` element will appear. The `div` will have a light green background and padding.

4. Dynamic Attributes and Events

This example shows how to dynamically add attributes and event listeners to elements.

const button = createElement({ tag: 'button', attributes: { id: 'myButton' }, children: 'Click Me' // Button text }); button.addEventListener('click', () => { alert('Button clicked!'); // Alert on button click }); appendToDOM({ element: button, parent: 'body' });

Output: A button labeled "Click Me" will appear. When clicked, it will show an alert that says "Button clicked!"

5. Applying Styles

Example of applying CSS styles to a specific selector.

applyStyles({ selector: '#myDiv', // Targeting div with id myDiv css: { backgroundColor: 'yellow', fontSize: '20px', textAlign: 'center' // Apply styles like background color and text alignment } });

Output: The div with the ID `myDiv` will have a yellow background, font size set to 20px, and its text will be centered.

6. Creating a Table

This example demonstrates how to create a simple HTML table with rows and cells.

const table = createElement({ tag: 'table', children: [ createElement({ tag: 'tr', children: [ createElement({ tag: 'th', children: 'Header 1' }), createElement({ tag: 'th', children: 'Header 2' }) ]}), createElement({ tag: 'tr', children: [ createElement({ tag: 'td', children: 'Row 1, Cell 1' }), createElement({ tag: 'td', children: 'Row 1, Cell 2' }) ]}), createElement({ tag: 'tr', children: [ createElement({ tag: 'td', children: 'Row 2, Cell 1' }), createElement({ tag: 'td', children: 'Row 2, Cell 2' }) ]}) ] }); appendToDOM({ element: table, parent: 'body' });

Output: A table with headers "Header 1" and "Header 2", and two rows containing data such as "Row 1, Cell 1" and "Row 2, Cell 1" will be created and displayed.
